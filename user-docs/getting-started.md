# Memulai - Panduan Pengguna RMS

Panduan ini akan membantu Anda memulai dengan sistem RMS (Repair Management System).

## Persiapan Awal

Sebelum menggunakan sistem RMS, pastikan Anda telah:

- ✅ Memiliki akun pengguna yang terdaftar
- ✅ Mengetahui email dan kata sandi Anda
- ✅ Memiliki koneksi internet yang stabil
- ✅ Menggunakan browser web modern (Chrome, Firefox, Safari, atau Edge)

---

## Cara Masuk ke Sistem

### Langkah-langkah Login

1. **Buka halaman login**
   - Navigasi ke URL aplikasi RMS di browser Anda

2. **Masukkan kredensial**
   - Masukkan **email** yang terdaftar
   - Masukkan **kata sandi** Anda

3. **Klik tombol "Masuk"**
   - Sistem akan memverifikasi kredensial Anda

4. **Akses dashboard**
   - Setelah berhasil login, Anda akan diarahkan ke dashboard sesuai peran Anda

### Masalah Login Umum

| Masalah | Solusi |
|---------|--------|
| Email tidak ditemukan | Pastikan email yang dimasukkan benar, hubungi admin jika perlu |
| Kata sandi salah | Periksa keyboard, pastikan Caps Lock tidak aktif |
| Akun tidak aktif | Hubungi administrator sistem |

---

## Mengenal Dashboard

Setelah login, Anda akan melihat dashboard yang berbeda tergantung peran Anda:

### Dashboard Admin
- Ringkasan semua toko
- Statistik sistem secara keseluruhan
- Akses ke manajemen pengguna dan inventaris

### Dashboard Staff
- Statistik layanan toko
- Daftar layanan terbaru
- Akses cepat untuk membuat layanan baru

### Dashboard Teknisi
- Tugas yang tersedia
- Layanan yang sedang dikerjakan
- Riwayat perbaikan

---

## Navigasi Sistem

### Sidebar Navigasi

Sidebar di sebelah kiri menyediakan akses ke semua fitur:

| Menu | Deskripsi |
|------|-----------|
| 🏠 **Beranda** | Kembali ke dashboard utama |
| 📊 **Dashboard** | Lihat statistik dan ringkasan |
| 🔧 **Layanan** | Kelola layanan perbaikan |
| 📦 **Inventaris** | Kelola suku cadang dan harga |
| 👥 **Karyawan** | Manajemen pengguna (Admin only) |
| 🏪 **Toko** | Kelola data toko (Admin only) |

### Menu Pengguna

Klik pada avatar/nama Anda di pojok kanan atas untuk:
- Melihat profil Anda
- Mengatur preferensi
- Keluar dari sistem

---

## Pengaturan Profil

### Mengubah Kata Sandi

1. Buka menu pengguna (pojok kanan atas)
2. Pilih "Pengaturan"
3. Pilih "Ubah Kata Sandi"
4. Masukkan kata sandi lama dan baru
5. Simpan perubahan

### Tips Keamanan

- Gunakan kata sandi yang kuat (minimal 8 karakter)
- Jangan bagikan kata sandi Anda
- Keluar dari sistem setelah selesai menggunakan
- Jangan gunakan komputer publik untuk akses sensitif

---

## Peran Pengguna

Sistem RMS memiliki 3 peran pengguna:

### 👑 Admin
- Akses penuh ke semua fitur
- Dapat mengelola semua toko
- Dapat menambah dan mengelola pengguna
- Dapat melihat data dari semua toko

### 📝 Staff
- Dapat membuat layanan baru
- Dapat mengelola informasi pelanggan
- Dapat melihat semua layanan di tokonya
- Dapat menambah pengguna ke tokonya

### 🔧 Teknisi
- Dapat melihat layanan yang tersedia
- Dapat mengambil tugas perbaikan
- Dapat update status layanan
- Dapat menambah suku cadang dan layanan ke perbaikan
