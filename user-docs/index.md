# RMS - Panduan Pengguna Sistem Manajemen Perbaikan

Selamat datang di Panduan Pengguna RMS (Repair Management System). Dokumentasi ini menyediakan panduan lengkap untuk semua peran pengguna dalam sistem.

## Pengantar

RMS adalah Sistem Manajemen Perbaikan yang komprehensif yang dirancang untuk toko perbaikan ponsel untuk mengelola:

- **Layanan** - Lacak layanan perbaikan dari penerimaan hingga selesai
- **Inventaris** - Kelola suku cadang dan daftar harga layanan
- **Operasi Multi-Toko** - Tangani beberapa lokasi toko
- **Manajemen Pengguna** - Kontrol akses berbasis peran

## Persyaratan Sistem

- Browser web modern (Chrome, Firefox, Safari, Edge)
- Koneksi internet
- Akun pengguna yang valid dengan peran yang ditetapkan

## Cara Masuk

1. Navigasi ke URL aplikasi RMS
2. Masukkan email dan kata sandi Anda
3. Klik "Masuk"
4. Anda akan diarahkan ke dashboard khusus peran Anda

> **Catatan:** Jika Anda mengalami masalah masuk, hubungi administrator sistem Anda atau lihat bagian [FAQ](faq.md).
